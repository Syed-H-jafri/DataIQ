# DataIQ

Intelligent data matching desktop application using Flutter and FastAPI.

## Overview
DataIQ helps users automatically match and compare data from different file formats such as Excel, CSV, and PDF. It reduces manual effort and improves accuracy by using smart comparison techniques.

## Features
- Supports multiple file formats
- Automatic matching with confidence scores
- Fast and easy-to-use interface
- Desktop application with installer support

## Technologies Used
- Flutter (Frontend UI)
- FastAPI (Backend API)
- Python (Data processing and matching logic)

## How to Use
1. Download the setup file from the Releases section
2. Install and open the application
3. Upload original and comparison files
4. Click "Upload & Match"
5. Download the results

## Challenges Faced
- Packaging Python backend into executable (api.exe)
- Handling communication between Flutter and FastAPI
- Managing API startup timing in desktop environment
- Resolving Windows security and execution issues
